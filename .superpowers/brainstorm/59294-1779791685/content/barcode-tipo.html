¿Qué tipo de código tienen los albaranes?

Necesito saber qué datos contiene el código para definir si podemos generar el PDF localmente o necesitamos consultar al proveedor

Code 128 / Code 39 / EAN-13

Código de barras lineal

Solo contiene un número (p.ej. el número de albarán o código de artículo). Para generar el PDF habría que buscar los datos en el portal del proveedor (Hefame, BIDA...).

QR Code

QR Code

Puede contener texto largo, URLs, o datos estructurados (JSON, XML). Algunos proveedores incluyen todos los datos del albarán en el QR. Potencialmente no necesita API externa.

PDF417 / DataMatrix

Código 2D de alta densidad

PDF417 o DataMatrix: común en albaranes farmacéuticos. Puede contener varias líneas de productos, cantidades y precios directamente codificados. Ideal para generar PDF sin API.
